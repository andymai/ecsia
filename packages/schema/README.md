# @ecsia/schema

The type-level vocabulary for [**ecsia**](https://github.com/andymai/ecsia) — a fast,
type-safe Entity Component System for TypeScript.

`@ecsia/schema` provides the field tokens and the 1..8 arity query inference that the kernel
and umbrella build on. It is almost entirely compile-time machinery.

> **Status:** 0.1.0, unpublished. Most users want the umbrella package
> [`ecsia`](https://www.npmjs.com/package/ecsia), which re-exports everything you need. This
> package is pulled in transitively; you rarely depend on it directly.

## Install

```sh
pnpm add @ecsia/schema   # not yet published — local workspace for now
```
